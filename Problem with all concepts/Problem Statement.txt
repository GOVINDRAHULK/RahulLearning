Write a C# program to perform FILE I/O operations for maintaining the students records with the details such as Name, Register Number, Course, two subject marks, Total marks, Average, Grade. Here, must collect the necessary inputs such as Name, Register Number, Course and Two subject marks from user and write into a file. The same information can be retrieved from the same file and process it for finding the Total Marks, Average, and Grade. Then, resulted values must be updated into the same file. Finally, display all the student details in proper format.
